| Q-EFF-09 | 行動単位効果期間の減算と失効         | 付与された当該行動では減算しない。次回以降の対象自身の行動終了時に1減らし、0になった時点で失効する。                                                                           |
| Q-EFF-10 | 重複効果の期間管理                   | 重複あり・重複なしのどちらも、効果インスタンスと効果期間を個別に保持する。再付与によって既存インスタンスの期間を上書きしない。                                                 |
| Q-EFF-11 | 重複なし効果の繰り上げ               | 現在有効な最強効果が失効した場合、残っている同種効果から次に強い効果を即時に有効化する。                                                                                       |
| Q-EFF-12 | ターン単位効果期間の減算と失効       | 付与されたターンの終了時には減算しない。次のターン終了時から1減らし、0になった時点で失効する。                                                                                 |

## 管理規則

- 新たな不明点が見つかった場合は、一意な論点IDを付けて「確認待ち事項」または「保留事項」へ追加する。
- 保留事項が確定した場合は「決定事項」の該当領域へ移動する。
- 決定内容を変更する場合は、ユビキタス言語、ユースケース、境界づけられたコンテキストへの影響を同時に確認する。
- 保留事項に依存する実装では振る舞いを推測せず、仕様確定まで差し替え可能なポリシーまたは未対応として扱う。
